Integrates Qody's Automation into your site, giving it that competative, stupidly crazy done-for-you boost of awesome.

== Description ==

Qody is the one-stop shop for all things automated for web-based businesses. Everything from custom scripts, Wordpress plugins, rank tracking, request handling, article managing, to backlink organizing, accounting, social media, calculating and more.

If it's done manually, chances are Qody can automate it.

For more information about Qody's work and his latest accomplishments, advice, and news, head on over to http://qody.co for the latest scoop.

== Installation ==

1. Upload `qodys-connector` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the "Qody" menu under the dashboard section in the top-left corner of the Wordpress admin area
